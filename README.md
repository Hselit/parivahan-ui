## 📌 Overview

This project is a UI clone of the VAHAN 4.0 Citizen Services homepage built using React 18 and Vite.  
It focuses on responsive design, multilingual support (English & Hindi), and a modular feature-based architecture.

---

## 🛠 Tech Stack

- React 18 (Vite)
- react-i18next + i18next (multilingual support)
- Pure CSS (no UI library used)
- Vite (fast development & build tool)
